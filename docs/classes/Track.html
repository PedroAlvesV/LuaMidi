Track:new ([name])
Creates a new Track

Parameters:

  • name string a name metadata to the Track (optional)

Returns:

    new Track object
Track:add_event (events, map_function)
Adds an event-list (or single event) to the track. These events can be MetaEvents, NoteEvents or ProgramChangeEvents.

Parameters:

  • events a single event or event-list (array of events).
  • map_function function to be applied to events (being it an array or not).

Returns:

    new Track object

See also:

  • MetaEvent
  • NoteEvent
  • ProgramChangeEvent
Track:get_events (filter)
Gets events from Track

Parameters:

Returns:

Track:set_time_signature (num, den[, midi_clocks_tick=24[, notes_midi_clock=8]])
Sets Track's time signature

Parameters:

  • num int signature's numerator (top number)
  • den int signature's denominator (bottom number)
  • midi_clocks_tick number of MIDI clocks per ticks (default 24)
  • notes_midi_clock number of notes per MIDI clock (default 8)

Returns:

    Track with time signature

See also:

    MetaEvent
Track:set_key_signature (sf, mi)
