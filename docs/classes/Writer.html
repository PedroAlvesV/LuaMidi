Writer:build_file ()
Concatenates everything to an array. This array is must be unpacked and translated to binary to produce a MIDI file.

Note: This function should not be invoked by the user. It's purpose is debugging LuaMidi.

Returns:

    builded array

See also:

    save_MIDI
